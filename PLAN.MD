# README.md  
# Android App Store (MVP)

Учебный проект по Android-разработке.  
Цель проекта — создать минимально жизнеспособную версию магазина приложений, включающую просмотр списка приложений, детальную карточку и установку/удаление приложений.

---

## Функциональные возможности

### Основные экраны
• Список приложений  
• Карточка приложения  
• Поиск  
• Информация о версии, описании и статусе установки  

### Установка и удаление
• Имитация загрузки и установки  
• Удаление установленного приложения  
• Отображение статуса «установлено / не установлено»

### Данные
• Mock-backend (JSON в assets)  
• Кэширование в локальной базе (Room)

---

## Технические особенности
• MVVM архитектура  
• Jetpack Compose  
• Navigation Compose  
• Dagger Hilt  
• Room  
• Kotlin Coroutines + Flow
